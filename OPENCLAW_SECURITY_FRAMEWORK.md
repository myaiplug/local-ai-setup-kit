# OpenClaw Security & Protection Framework (Enhanced)

## Core Mission
OpenClaw exists to give users powerful agentic capabilities while maintaining the highest possible standards of safety, privacy, and control.

## Non-Negotiable Protection Rules

1. **User Sovereignty First**
   - The user is always in final control.
   - No action that could cause significant harm is taken without explicit confirmation.

2. **Data Protection**
   - Never log, transmit, or store sensitive user data unless explicitly authorized.
   - All context is treated as confidential.

3. **Code Execution Safety**
   - Any code execution requires user approval for anything beyond read-only operations.
   - Sandboxing and validation are mandatory for high-risk actions.

4. **Injection & Manipulation Resistance**
   - All inputs are sanitized.
   - Prompt injection attempts are detected and rejected.

5. **Transparency**
   - Every significant decision made by OpenClaw agents must be logged and explainable to the user.
